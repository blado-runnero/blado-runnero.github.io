How to setup a google cloud instance for deep learning??

 

 

 

  1. Go to google cloud and login/sign up and open google console.

https://console.cloud.google.com

  1. Go to Billings and check if you have a free credit or about 20k inr or not.

Yes? --> Go ahead

No ? --> Make new account and get free credits. [ Requires a credit/debit card ]

  1. Go to this link and click on Launch on Compute Engine.

  2. You will be prompted a screen to ask which project its for?, give desirable project name or choose any one if you already have one.

  3. Now is the time for choosing specs for you vm,

    The default option is good, I am suggesting 2 changes for more efficient instance.

    • Zone :- us-west1-b GPU type :- NVIDIA TESLA V100 Number of GPUs :- 1 (1 is giving me max 80s for each epoch, you can try another combo for more faster speed, I haven't tried it yet)
    • Tick the following option :- Install NVIDIA GPU driver automatically on first startup?

    Click on Deploy (This can take upto 5 mins)

  4. Now go to your Compute Engine --> VM instance and check if its running or not, it should be ideally running and if not then start it. After that type "Firewall Rules", Select the one in VPC network. Make a new entry, set targets to "All instance in the network", and in source ip ranges, fill 0.0.0.0/0 , and in protocol tick tcp and fill the following in the text box, 5000,5001,5002,5003,5004,5005,5006. and click on create.

  5. Now see on the left sidebar, search External IP Addresses and open it. Find the External address of your current instance and make its Type as Static.

  6. Go to your instance, Compute Engine --> VM instance . Make an ssh connection, by clicking on connect and choosing ssh option. An command prompt window will appear. If it asks for any permission for installing CUDA driver or anything related to it, enter y for YES.

  7. Enter following commands

All this commands needs to be done without any interruptions, otherwise it will take a lot time to fix later.

  • pip install jupyter notebook
  • pip install keras
  • pip install tensorflow
  • pip install tensorflow-gpu

 

Now time to setup jupyter for our connection.

  • jupyter notebook --generate-config

This will save the config to an address, go to that address by using cd command. e.g. cd home , cd <username> , cd .jupyter

  • nano <file_name_of_config>
  • paste the following code at once on the screen and make sure none is commented.

c = get_config() c.NotebookApp.ip = '0.0.0.0' c.NotebookApp.open_browser = False c.NotebookApp.port = 5003

  • Press Ctrl+O , then Enter. And finally Ctrl+X to come out of it.
  • jupyter notebook password { make any password }

Setting of jupyter and ports are done.

 

  1. Now you can run jupyter by following command
  • jupyter notebook --no-browser --port=5003

This ideally should show no errors.

  1. Now Go to Compute Engine --> VM instance, copy external IP. Open the browser and open
  • http://<external IP>:5003 e.g. http://35.184.210.56:5003

You should be asked a password and you can enter the one that you entered earlier.

 

 

Tips :-

  • The port number should be same on all the lines, i.e. in firewall, jupyter config file, and in step 10.

  • Stop the instance after your work, its charged hourly and you will run out of credits in 7-8 days, and will be charged.

  • For uploading and downloading the data, use ssh command line's options tab.

  • For further processing , save weights after the training.

  • Use https://www.hostingmanual.net/zipping-unzipping-files-unix/ this link for zipping and downloading multiple files.
